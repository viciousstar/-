需求:	- Microsoft Visual Basic 6 (或SP3补丁包)
	- 请把文件夹TextCompress下的TextCompress.exe拷贝到你的系统目录下，如 \windows\system32

功能:
---------------
	- 利用哈夫曼编码压缩文件
	- 或者解压利用本程序压缩的后缀名为wbys的文件

安装:
---------------
	- 打开TextCompressUI.exe即可

怎样使用:
---------------
 	- 打开目标文件
	- 查看可以调用记事本进行查看
	- 选择进行压缩, 或者解压缩, 解压缩只支持后缀为wbys格式的文件, 如果不是会发生错误, 程序没有错误处理, 请慎重
技术支持:
---------------
	- http://www.cnblogs.com/this-543273659/archive/2011/07/30/2122246.html 博客园puresky关于哈夫曼编码c语言实现
	- http://www.codesky.net/showhtml/16321.htm 仿vb版winrar
