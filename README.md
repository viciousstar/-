# SE-Project-HIT
A project of software engineering of HIT.A simple cms website using django.
In this website,you can:

- create group
- join group
- create post in the group
- comments on the post
- message to somebody

## Dependences
 - python 3
 - django 1.8
 - bootstrap
 - django_avatar
 - django_messages
 - django-stronghold `登陆限制`
 - django-pagedown
 - markdown
